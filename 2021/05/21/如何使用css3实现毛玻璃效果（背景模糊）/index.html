如何使用css3实现毛玻璃效果（背景模糊）

#css3   

设计搞了一个背景模糊效果，一看内心一个mmp，但是这也难不倒我们这帮蓝领人才

filter

网上查了一下css3有一个filter的效果，可以实现背景模糊，上效果

上代码：

<!DOCTYPE html>
<html>

<head>
  <title>test</title>
  <style>
  html &#123;
    overflow: hidden;
  &#125;

  body &#123;
    width: 100vw;
    height: 100vh;
    margin: 0;
    position: relative;
  &#125;

  main &#123;
    width: 50%;
    height: 50%;
    position: absolute;
    top: 0;
    left: 0;
    bottom: 0;
    right: 0;
    margin: auto;
    border-radius: 5px;
    overflow: hidden;
  &#125;

  main::before &#123;
    content: '';
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    /* filter */
    filter: blur(20px);
    z-index: -1;
    margin: -30px;
  &#125;

  &#125;
  </style>
</head>

很好，这很完美的实现了模糊的效果，但是有亿点点弊端，那就是main要加背景图片才可以实现模糊的效果，那如果父盒子没有背景图呢？或者这个父盒子是个地图、canvas、svg呢？

欸，恭喜，还真有解决的方法

backdrop-filter
